# Arduino × Max 인터랙티브 미디어 워크숍

**멘토링 대상:** YBH  
**환경:** Arduino Uno R3 + Max/MSP 9 (일부 Max 8)  
**시리얼 통신:** 9600 baud, 커스텀 바이트 프로토콜

---

## 이게 뭔가요

아두이노 센서/액추에이터와 Max/MSP를 시리얼로 연결해서,  
물리적인 입력을 영상·사운드·3D 그래픽으로 바꾸는 프로젝트 모음입니다.

각 프로젝트는 **3단계 구조**로 되어 있습니다:

| 단계 | 이름 | 설명 |
|------|------|------|
| 01 | **Basic** | 센서/모터의 기본 통신 확인. "일단 값이 오는지 보자" |
| 02 | **Application** | 받은 값으로 뭔가 실용적인 걸 해봄 |
| 03 | **MediaArt** | 같은 센서로 미디어아트 작품 수준의 결과물 만들기 |

## 프로젝트 목록

### [Project 1 — 가변저항 (Potentiometer)](./Project1/)
가변저항 하나로 시작합니다. 돌리면 값이 바뀌고, 그 값으로 동영상 볼륨을 조절하거나 영상을 역재생시킵니다.

### [Project 2 — 초음파 센서 (HC-SR04)](./Project2/)
손을 가까이/멀리 하면 거리가 바뀝니다. 거리에 따라 사운드 필터가 열리고 닫히거나, 영상에 안개가 끼는 효과를 만듭니다.

### [Project 3 — 서보모터 (SG90)](./Project3/)
이번엔 반대 방향입니다. Max에서 아두이노로 값을 보내서 모터를 제어합니다. 메트로놈, 소리 반응 서보 등.

### [Project 4 — 조도센서 (Light Sensor)](./Project4/)
빛의 밝기를 읽습니다. 방이 어두워지면 화면도 어두워지고, 소리도 먹먹해지는 몰입형 미디어아트를 만듭니다.

### [Project 5 — 조이스틱 (Joystick)](./Project5/)
X축, Y축, 버튼 — 3개의 입력을 동시에 다룹니다. 음고+볼륨 제어부터 3D 큐브 회전까지.

## 시리얼 프로토콜

모든 프로젝트가 같은 규칙을 씁니다:

```
[주소 바이트] [값 바이트]
```

아두이노가 보내는 경우:
| 주소 | 센서 | 값 범위 |
|------|------|---------|
| 4 | 가변저항 (A4) | 0~255 |
| 5 | 초음파 거리 | 0(가까움)~255(멀음) |
| 6 | 조도센서 | 0(밝음)~255(어두움) |
| 10 | 조이스틱 X | 0~255 |
| 11 | 조이스틱 Y | 0~255 |
| 12 | 조이스틱 버튼 | 0 또는 1 |

Max가 보내는 경우 (서보모터):
- 0~180 사이의 1바이트 → 서보 각도

## Max 패치 공통 구조

거의 모든 Max 패치가 이 흐름을 따릅니다:

```
toggle(Open Port) → select 0 1 → close / open
                                        ↓
toggle(Start) → qmetro 30 ──→ serial c 9600
                                        ↓
                                   zl group 2
                                        ↓
                                   route [주소]
                                        ↓
                                   scale → ...
```

**주의:** `serial c 9600`에서 `c`는 포트 이름입니다. 본인 컴퓨터에 맞게 바꿔야 합니다.  
Mac이면 `serial usbmodem14101 9600` 같은 식으로요.

---

## 시작하기 전에

1. 아두이노 IDE에서 `.ino` 파일을 업로드
2. Max에서 `.maxpat` 파일을 열기
3. `serial` 오브젝트의 포트 이름을 본인 환경에 맞게 수정
4. **Open Port** 토글 → **Start** 토글 순서로 켜기
5. 끌 때는 반대 순서: Start 끄기 → Open Port 끄기

시리얼 포트가 안 잡히면 Max 콘솔에 에러가 뜹니다. 아두이노 IDE의 시리얼 모니터가 열려 있으면 포트를 점유하고 있어서 Max에서 못 열어요. 모니터 먼저 닫아주세요.
