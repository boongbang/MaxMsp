# Project 1 — 가변저항 (Potentiometer)

가변저항은 가장 단순한 아날로그 입력입니다.  
돌리면 0~1023 사이의 값이 나오고, 이걸 0~255로 줄여서 Max에 보냅니다.

---

## 배선

```
가변저항 3핀:
  왼쪽  → GND
  가운데 → A4
  오른쪽 → 5V
```

딱 3개 선이면 끝입니다.

---

## 프로토콜

```
[주소: 4] [값: 0~255]
```

50ms 간격으로 계속 보냅니다 (초당 20회).

---

## 01. Basic — 기본 통신

**목표:** 가변저항을 돌리면 Max에서 숫자가 바뀌는 걸 확인

아두이노 쪽은 진짜 간단합니다. `analogRead(A4)`로 읽고, `map`으로 0~255 범위로 맞추고, `Serial.write`로 보내면 끝.

Max 쪽에서는:
- `serial` → `zl group 2` → `route 4` → `number`
- `route 4`가 주소 바이트를 걸러서 값만 꺼내줍니다
- 마지막에 `print`로 Max 콘솔에도 찍어봅니다

**확인 방법:** 가변저항을 돌리면 number 박스의 숫자가 0~255 사이에서 부드럽게 바뀌면 성공.

---

## 02. Application — 동영상 볼륨 조절

**목표:** 가변저항으로 `jit.movie`의 볼륨을 실시간 조절

아두이노 코드는 01과 완전히 동일합니다. 달라지는 건 Max 쪽뿐.

추가된 부분:
- `scale 0 255 0. 1.` — 센서값을 0.0~1.0 볼륨 범위로 변환
- `vol $1` 메시지 — `jit.movie`에 볼륨 명령을 보냄
- `jit.movie 320 240` + `jit.pwindow` — 영상 재생 및 표시

**사용 순서:**
1. Open Port
2. Start
3. Render (영상 렌더링 시작)
4. `read` 메시지 클릭해서 동영상 파일 선택

가변저항을 끝까지 돌리면 소리가 최대, 반대로 돌리면 무음.

---

## 03. MediaArt — 타임스트레치 왜곡

**목표:** 가변저항 위치에 따라 영상이 역재생/슬로모션/빨리감기

여기서도 아두이노 코드는 동일합니다. Max에서 매핑만 바뀝니다:

- `scale 0 255 -2. 2.` — 재생속도(rate)를 -2.0 ~ 2.0으로 매핑
- `rate $1` 메시지를 `jit.movie`에 전달

| 가변저항 위치 | rate 값 | 결과 |
|--------------|---------|------|
| 왼쪽 끝 | -2.0 | 2배속 역재생 |
| 중간쯤 왼쪽 | -1.0 | 정상속도 역재생 |
| 정중앙 | 0.0 | 정지 |
| 중간쯤 오른쪽 | 1.0 | 정상 재생 |
| 오른쪽 끝 | 2.0 | 2배속 재생 |

`crashtest.mov`는 Max에 내장된 샘플 영상입니다. 다른 영상을 쓰고 싶으면 `read` 메시지의 파일명을 바꾸면 됩니다.

가변저항을 천천히 돌리면 영상이 테이프 되감기하듯 왜곡되는데, 오디오도 같이 늘어나고 줄어들어서 꽤 신기한 느낌이 납니다.
