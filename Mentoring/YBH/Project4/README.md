# Project 2 — 초음파 센서 (HC-SR04)

초음파를 쏘고 돌아오는 시간을 재서 거리를 측정합니다.  
손을 가까이/멀리 하는 것만으로 인터랙션이 가능해서, 비접촉 제스처 인터페이스로 쓸 수 있습니다.

---

## 배선

```
HC-SR04:
  VCC  → 5V
  Trig → 디지털 12번
  Echo → 디지털 13번
  GND  → GND
```

---

## 프로토콜

```
[주소: 5] [값: 0~255]
```

0 = 가까움 (2cm), 255 = 멀음 (200cm).  
20ms 간격으로 전송 (초당 50회).

---

## 아두이노 코드 특징

초음파 센서는 가변저항과 달리 노이즈가 꽤 심합니다. 그래서 두 가지 처리를 합니다:

1. **3회 측정 평균** — 매 루프마다 3번 측정해서 평균을 냄. 튀는 값을 줄여줍니다.
2. **스무딩** — `(이전값×3 + 새값×7) / 10`. 값이 갑자기 확 바뀌는 걸 막아줍니다.

유효 범위(2~200cm) 밖의 측정은 버리고, 이전 값을 유지합니다.

---

## 01. Basic — 기본 통신

**목표:** 거리값을 Max에서 확인

Max 패치에서 두 가지 형태로 보여줍니다:
- **raw 값 (0~255):** `route 5` 직후의 number 박스
- **cm 값 (2~200):** `scale 0 255 2 200`으로 역변환

둘 다 보여주는 이유는, 실제로 쓸 때는 raw 값이 편하지만 디버깅할 때는 cm가 직관적이기 때문입니다.

---

## 02. Application — 물속 사운드 필터

**목표:** 거리에 따라 소리가 물속에 잠긴 것처럼 바뀜

손이 멀어질수록:
- **로우패스 필터**가 닫힘 (5000Hz → 200Hz) — 고음이 사라지면서 먹먹해짐
- **리버브 딜레이**가 늘어남 (10ms → 200ms) — 울림이 깊어짐

이 두 효과가 합쳐지면 물속에 잠긴 듯한 사운드가 됩니다.

Max 시그널 체인:
```
cycle~ 440 (사인파)
    ↓
lores~ (로우패스 필터) ← line~ ← scale (센서→주파수)
    ↓
tapin~ / tapout~ (딜레이) ← scale (센서→딜레이타임)
    ↓
*~ 0.5 (피드백)
    ↓
+~ (드라이+웻 믹스)
    ↓
gain~ → ezdac~
```

**사용 순서:** Open Port → Start → ezdac~ 스피커 버튼 클릭

---

## 03. MediaArt — 안개 영상 + 리버브 음악

**목표:** 거리에 따라 영상에 안개가 끼고 + 음악이 물속 느낌으로

P2_3은 Max 패치 파일이 문서에는 포함되어 있지 않지만, 개념적으로 02의 사운드 필터와 Project 4-3의 영상 밝기/필터를 결합한 형태입니다.

거리가 멀어지면:
- 영상: 밝기가 낮아지고 블러 효과 → 안개 속 느낌
- 오디오: 로우패스 필터 + 리버브 → 먹먹하고 울리는 사운드

하나의 센서 값에서 여러 파라미터를 동시에 매핑하는 것이 핵심입니다.  
같은 0~255를 밝기·필터 주파수·리버브 양에 각각 다른 범위로 `scale`합니다.
