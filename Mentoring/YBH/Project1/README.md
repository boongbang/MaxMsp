# Project 5 — 조이스틱 (Joystick)

지금까지는 센서 하나에서 값 하나를 보냈는데,  
조이스틱은 **X축, Y축, 버튼** 3개의 값을 동시에 보냅니다.  
멀티 채널 통신이 처음 등장하는 프로젝트입니다.

---

## 배선

```
조이스틱 모듈:
  GND  → GND
  +5V  → 5V
  VRx  → A0
  VRy  → A1
  SW   → 디지털 2번 (내부 풀업 사용)
```

---

## 프로토콜

한 루프에 3쌍을 연속으로 보냅니다:

```
[10][X값 0~255]
[11][Y값 0~255]
[12][버튼 0 or 1]
```

30ms 간격. 주소가 10, 11, 12로 다르기 때문에 Max에서 `route 10 11`로 분리합니다.

---

## 01. Basic — 기본 통신

**목표:** X, Y 값이 따로따로 표시되는지 확인

Max에서의 핵심:
- `route 10 11` — 3개 outlet 중 첫째가 X, 둘째가 Y
- 버튼(주소 12)은 이 패치에서는 사용하지 않아서 route의 세 번째 outlet으로 빠짐

조이스틱을 움직이지 않으면 X, Y 모두 127~128 근처 (중앙값).  
왼쪽으로 밀면 X가 줄고, 오른쪽으로 밀면 X가 늘어납니다.

---

## 02. Application — 음고(X) + 볼륨(Y)

**목표:** 조이스틱으로 테레민처럼 소리를 조절

X축 (좌우) = 음고:
- `scale 0 255 220. 880.` — 라3(220Hz)부터 라5(880Hz)까지
- `line~`을 통해 부드럽게 변화 → `cycle~`의 주파수로 들어감

Y축 (위아래) = 볼륨:
- `scale 0 255 0. 1.` — 위가 1.0(크게), 아래가 0.0(작게)
- `line~`을 통해 → `*~`로 사인파에 곱해서 볼륨 조절

```
cycle~ (주파수 ← X축)
    ↓
*~ (볼륨 ← Y축)
    ↓
gain~ → ezdac~
```

조이스틱을 비스듬히 밀면 음고와 볼륨이 동시에 바뀝니다.  
`$1 20` 메시지로 20ms 글라이드를 넣어서 음이 끊기지 않고 미끄러지듯 바뀝니다.

---

## 03. MediaArt — 3D 큐브 회전

**목표:** 조이스틱으로 3D 큐브를 실시간 회전

이 프로젝트는 아두이노 코드가 이전과 다릅니다.  
**아두이노가 직접 각도를 누적 계산합니다:**

```cpp
angleY += speedY * 3.0;  // X축 입력 → Y축 회전
angleX -= speedX * 3.0;  // Y축 입력 → X축 회전
```

조이스틱을 기울이면 그 방향으로 계속 회전하고, 중앙에 놓으면 멈춥니다.  
데드존(`abs < 0.15`)을 넣어서 손을 떼도 미세하게 돌아가지 않게 했습니다.

각도는 0~360도 범위를 유지하면서 0~255로 매핑해서 전송합니다.

Max 쪽 — Jitter 3D 렌더링:
- `jit.gl.gridshape ctx @shape cube` — 큐브 생성
- `jit.gl.light ctx` — 조명 (없으면 입체감이 안 나옴)
- `jit.gl.render ctx` — 렌더링
- `jit.window ctx @size 640 480` — 창 표시

렌더 루프:
```
qmetro 20 → t b b b → erase(화면 지우기)
                     → cube에 bang(그리기)
                     → render에 bang(출력)
```

`rotatexyz $1 $2 $3` 메시지로 X, Y, Z 세 축 회전을 한번에 지정합니다.  
Z축은 0으로 고정이고 X, Y만 조이스틱으로 제어합니다.

`@erase_color 0.12 0.12 0.18 1.`은 배경색인데, 약간 어두운 남색입니다.  
큐브 색상 `0.4 0.65 1. 1.`은 하늘색 계열이라 배경과 잘 어울립니다.

**사용 순서:** Open Port → Start → Render (세 토글 모두 켜기)
